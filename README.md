知识图谱是由一些相互连接的实体和他们的属性构成的。

表示为一个**SPO**三元组(Subject-Predicate-Object)，

常用用**RDF** (Resource Description Framework) 形式化地表示这种三元关系。所有的实体都能这样表达，抽象程度可以很高，好的知识图谱能非常合适的根据场景控制粒度，所有的属性直指实质。
